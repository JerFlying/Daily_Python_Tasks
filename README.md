## [Tool] TProuter_reboot

A simple script to reboot a typical TP-Link router. 

一次因为路由器卡顿造成广域网比局域网速度快，发现原因是路由器运行时间过长导致的。Archer C7没有定时重启功能，想到写一个路由器重启脚本。
